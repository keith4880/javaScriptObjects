Inheritance

Subclass and Superclass

Subclass - (child), class that inherits from another class.

Superclass - (parent), class being inherited from.

class Vehicle {

protected String brand = "Toyota"; // Vehicle Attribute
public void honk() { // Method
System.out.println("Tuut, tuut!");
}
}
class Car extends Vehicle {
private String modelName = "Camry"; //Car Attribute
public static void main(String[] args) {
Car myFastCar = new Car(); //New myFastCar object
myFastCar.honk(); //Calls honk() method

System.out.println(myFastCar.brand + " " + myFastCar.modelName);
}
}
